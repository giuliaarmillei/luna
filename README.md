# About `luna`

Questo pacchetto è stato creato a lezione 

Il pacchetto si installa così:
```r
library(devtools)
install_github("giuliaarmillei/luna")
```

Per usare il pacchetto va richiamato con la funzione `library()`
```r
library(luna)
```
# Struttura del pacchetto
Il pcchetto `luna` è organizzato nel modo seguente:

```r
luna/
├── DESCRIPTION
├── README.md
├── R/
│   ├── numero.R
│   └── somma.R
└── man/
    ├── numero.Rd
    └── somma.Rd
```
